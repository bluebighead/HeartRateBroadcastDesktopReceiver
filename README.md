# 心率广播接收软件

## 项目简介

心率广播接收软件是一款基于Python和PyQt5开发的桌面应用程序，用于接收和显示来自蓝牙设备（如智能手表、心率带等）的实时心率数据。该软件提供了直观的用户界面，支持心率数据记录、可视化、卡路里估算、心率变异性(HRV)计算以及OBS直播集成等功能。

## 主要功能

### 1. 心率数据接收与显示
- 实时接收并显示心率数据
- 心率动画效果，随心率变化
- 运动时长记录

### 2. 数据记录与统计
- 记录每一次心率变化及时间戳
- 查看历史记录
- 生成心率变化折线图
- 统计最大/最小/平均心率

### 3. 健康分析
- 卡路里消耗估算（基于体重、年龄、性别）
- 心率变异性(HRV)计算与评估
- HRV状态评估（良好、一般、较低）

### 4. OBS直播集成
- 支持文本文件和HTML文件输出
- 自定义心率图标（支持图片和GIF）
- 响应式设计，适应不同大小的显示区域

### 5. 其他功能
- 检查更新（支持自动下载和安装）
- 数据导出（CSV/Excel格式）
- 断连自动重连机制

## 技术栈

- **编程语言**: Python 3.7+
- **GUI框架**: PyQt5
- **蓝牙通信**: bleak
- **数据可视化**: pyqtgraph
- **数据处理**: numpy, pandas, openpyxl

## 安装与使用

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行程序

```bash
python main.py
```

### 打包成可执行文件

```bash
pyinstaller --onefile --windowed main.py
```

生成的可执行文件位于 `dist` 目录中。

## 使用说明

### 基本使用
1. 开启您的蓝牙心率设备（如智能手表）
2. 点击"开始接收"按钮
3. 软件会自动扫描并连接心率设备
4. 实时显示心率数据和运动时长

### 心率变异性(HRV)功能
1. 点击顶部菜单栏的"设置"
2. 选择"显示心率变异性（HRV）"
3. 首页会显示HRV数据，包括RMSSD值和状态评估

### OBS直播集成
1. 点击顶部菜单栏的"设置"
2. 选择"对接OBS"
3. 启用OBS对接功能
4. 选择输出方式（文本文件或HTML文件）
5. 设置数据文件保存路径
6. 选择自定义心率图标（仅HTML输出方式）
7. 在OBS中添加相应的源：
   - 文本文件：添加文本源，指向生成的txt文件
   - HTML文件：添加浏览器源，指向生成的HTML文件

## 常见问题

### 无法找到心率设备
- 确保蓝牙设备已开启并处于广播模式
- 确保设备与电脑的距离在蓝牙有效范围内
- 尝试重启蓝牙设备和电脑

### 心率数据不更新
- 检查设备是否仍在广播心率数据
- 尝试重新开始接收
- 检查电脑蓝牙连接状态

### OBS中显示异常
- 确保选择了正确的输出方式
- 检查文件路径是否正确
- 对于HTML输出，确保OBS浏览器源设置正确
